# Project-Game-Super-Mario
Pemain harus mengendalikan Mario untuk bergerak dari titik awal hingga mencapai tujuan akhir di setiap level. Dalam perjalanan tersebut, pemain akan menemukan banyak tantangan seperti monster/enemy, jebakan, dan area berbahaya lainnya dalam game. Karakter akan mengumpulkan koin sebanyak banyaknya.
